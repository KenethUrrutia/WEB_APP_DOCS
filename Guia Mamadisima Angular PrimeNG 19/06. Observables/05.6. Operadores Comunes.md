## 1. `map()`

Transforma cada valor emitido.

```ts

of(1, 2, 3).pipe(
  map(x => x * 2)
).subscribe(console.log);
```

**Salida:**

```

4
6
```

## 2. `filter()`

## 3. `take(n)`

Toma los primeros `n` valores y completa.

```ts
import { interval, take } from 'rxjs';

interval(1000).pipe(
  take(3)
).subscribe(val => console.log('tick:', val));
```

**Salida (cada segundo):**

```
tick: 0
tick: 1
tick: 2
```

Luego se completa autom√°ticamente.

## 4. `switchMap()`

Cancela el observable anterior si llega un nuevo valor. √ötil para peticiones HTTP reactivas.


- Si uno nuevo llega, **cancela el anterior**

## 7. `tap()`

Permite ejecutar efectos secundarios sin afectar el flujo.

```ts

  tap(val => console.log('Antes del map:', val)),
  map(val => val * 10)
).subscribe(console.log);
```
